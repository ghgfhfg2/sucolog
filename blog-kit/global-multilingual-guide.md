# 글로벌 페이지 확장 가이드 (서브디렉토리: EN/JA)

작성일: 2026-03-07

## 1) 목표
- 기존 한국어 블로그를 유지하면서 글로벌 유입 확대
- 1차 언어: **영어(en)**, **일본어(ja)**
- 구조: `example.com/en/`, `example.com/ja/`

## 2) 왜 이 방식인가
- 새 도메인 분리보다 초기 SEO/운영 리스크가 낮음
- 기존 도메인 신뢰도/링크 자산 활용 가능
- `hreflang`/내부링크/분석 관리를 한 곳에서 처리 가능

## 3) 콘텐츠 운영 원칙
1. **상위 성과 글부터 선별 번역** (전체 일괄 번역 금지)
2. **기계번역 100% 복붙 금지**
   - AI 번역은 초안으로만 사용
   - 제목/도입/CTA/예시는 사람 기준으로 재작성
3. **언어별 검색의도 맞춤**
   - 한국어 키워드 직역 금지
   - EN/JA는 별도 키워드 리서치 후 제목/H2 재설계

## 4) 기술 SEO 체크리스트
- [ ] 언어별 URL 생성: `/en/`, `/ja/`
- [ ] 각 문서에 `hreflang` 상호 연결 (`ko`, `en`, `ja`, `x-default`)
- [ ] 언어별 메타타이틀/메타디스크립션
- [ ] 언어별 XML 사이트맵
- [ ] 언어 선택 UI 제공 (강제 리다이렉트 최소화)
- [ ] Search Console 속성/보고서 언어별 점검

## 5) 작업 우선순위 (첫 4주)
### Week 1
- 상위 트래픽 글 20개 추출
- 번역 우선순위 5~10개 선정
- EN/JA 키워드 1차 맵 작성

### Week 2
- EN 5개 게시 (현지화 완료본)
- JA 3~5개 게시
- 내부링크/관련글 연결

### Week 3
- CTR 낮은 문서 타이틀/메타 A/B 개선
- FAQ/요약 섹션 보강

### Week 4
- 성과 리뷰(노출/클릭/체류/전환)
- 다음 배치(추가 5~10개) 선정

## 6) KPI (언어별 분리 측정)
- 노출수/클릭수/CTR
- 평균 순위
- 페이지 체류/이탈
- 전환(구독, 문의, 구매 등 목표에 맞게)

## 7) 확장 기준
- EN/JA가 2~3개월 안정적으로 성장하면 3번째 언어 검토
- 성과 검증 전 다언어(4개+) 동시 확장 금지

- 다음 액션: 대상 글 목록(5~10개) 확정 후 키워드 맵/초안 제작
