# 워드프레스 경제 블로그 초기 셋팅 가이드 (실행용)

## 0) 블로그 기본 방향
- 주 3회 발행 (월/수/금)
- 친근한 톤
- 상황극 비중 40%
- 중립적 관점 유지

---

## 1) 워드프레스 관리자 기본 셋팅

### 1-1. 설정 > 일반
- 사이트 제목: **(임시) 조카도 이해하는 오늘의 경제**
- 태그라인: **핫한 경제 이슈를 쉽게, 정확하게**
- 사이트 언어: 한국어
- 시간대: `서울`
- 날짜 형식: `Y년 m월 d일`
- 시간 형식: `H:i`

### 1-2. 설정 > 고유주소
- `글 이름` 선택
- 저장

### 1-3. 설정 > 토론
- 새 글에 댓글 허용: 켬
- 댓글 승인: 첫 댓글은 수동 승인 권장

---

## 2) 카테고리/태그 구조

### 2-1. 글 > 카테고리 (생성)
1. 오늘의 핫이슈
2. 삼촌의 용어 과외
3. 정책·금리·환율
4. 기업·산업 이슈
5. 주간 요약 리포트

### 2-2. 태그 운영 규칙
- 이슈성 태그: `FOMC`, `CPI`, `기준금리`, `환율`
- 산업 태그: `반도체`, `2차전지`, `AI`, `에너지`
- 태그는 글당 3~6개 유지

---

## 3) 필수 페이지 만들기

### 3-1. 페이지 생성
- 소개
- 문의
- 개인정보처리방침
- 면책 고지

### 3-2. 메뉴 연결
- 외모 > 메뉴에서 상단 메뉴 구성
- 푸터에도 개인정보처리방침/면책 고지 노출

---

## 4) 플러그인 최소 구성

1. SEO: Rank Math (또는 Yoast)
2. 보안: Wordfence
3. 캐시: LiteSpeed Cache (대안: WP Super Cache)
4. 백업: UpdraftPlus
5. 스팸 방지: Antispam Bee (대안: Akismet)

> 설치 후 즉시 업데이트 + 자동 업데이트 활성화 권장

---

## 5) 발행 워크플로우(주 3회)

- 월: 거시경제/정책
- 수: 기업/산업
- 금: 주간 요약 + 다음주 체크포인트

각 글 작성 순서:
1. 핫토픽 선정
2. 팩트 체크
3. 용어 2~4개 추출
4. 상황극 작성(40%)
5. 중립성 검수
6. 예약 발행

---

## 6) 고정 면책 문구 (글 하단)

본 글은 정보 제공 목적의 콘텐츠이며, 특정 금융상품 매수/매도를 권유하지 않습니다. 투자 판단의 책임은 본인에게 있으며, 최종 결정 전 공식 자료와 최신 공시를 반드시 확인하시기 바랍니다.

---

## 7) 품질 체크리스트
- [ ] 사실/의견/전망 분리
- [ ] 출처 확인 완료
- [ ] 상황극 비중 40% 반영
- [ ] 과장·선동 표현 없음
- [ ] 제목-본문 내용 일치
