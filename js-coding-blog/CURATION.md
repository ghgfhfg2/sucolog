# Curation Structure

이 문서는 JS Coding Blog의 문제 큐레이션 구조를 정의합니다.

## 3개 트랙

### 1. 오늘의 코테 (`today`)
목표:
- 도전 욕구 자극
- 매일 한 문제 도전 흐름 제공
- 초급/중급/상급 난이도 운영

특징:
- 신선함
- 짧지만 생각할 거리 있음
- 재방문 유도에 적합

### 2. 알고리즘별 코테 (`algorithm`)
목표:
- 개념 학습
- 유형별 반복 연습
- 초중급 실력 향상

특징:
- string, array, greedy, hash 등 주제별 분류
- 학습 흐름 중심 큐레이션

### 3. JS 메서드 학습 (`js-basic`)
목표:
- JavaScript 메서드 감각 익히기
- 초보자의 코테 진입장벽 낮추기

특징:
- 문자열/배열 메서드 중심
- 짧고 학습 친화적인 문제
- 알고리즘보다 언어 적응 중심

---

## 현재 샘플 문제 재분류
- `two-sum` → `today / warmup`
- `reverse-string` → `algorithm / string`
- `find-max` → `algorithm / array`
- `count-vowels` → `js-basic / string-methods`

---

## 추천 운영 방향
- 오늘의 코테는 꾸준한 리듬을 만드는 대표 콘텐츠로 운영
- 알고리즘별 코테는 실력 향상용 기본 축으로 운영
- JS 메서드 학습은 초급 유입과 친절한 입문용으로 운영
