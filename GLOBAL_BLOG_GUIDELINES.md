# GLOBAL_BLOG_GUIDELINES.md

> 글로벌(주제 공통) 블로그 운영 가이드라인

아래 원칙은 어떤 주제의 블로그에도 공통 적용한다.

## 1. 목적
- 장기적으로 신뢰받는 콘텐츠 자산 구축
- 지속 가능한 트래픽 유지/성장

## 2. 안전/윤리
- 불법·혐오·유해 콘텐츠 금지
- 저작권/개인정보/민감정보 보호

## 3. 품질
- 사실 기반 + 재현 가능성 확보
- 과장/낚시성 표현 최소화
- 실무 적용 가능한 정보 우선

## 4. SEO
- 제목 키워드 최적화(의도 중심)
- 명확한 본문 구조(H2/H3)
- 내부링크/메타/태그 일관성 유지
- 발행 전 금칙어·리스크 표현 점검

## 5. 운영
- 발행 후 성과 측정 및 업데이트 루프 유지
- 잘 되는 포맷은 확장, 약한 글은 개선

## 6. 오늘의 핫이슈 게시글(스토리텔링 확장 가이드)
- 매일 발행 시 먼저 당일 뉴스 흐름을 확인하고 주제를 1개 선정한다.
- 최소 2개 이상 신뢰 가능한 출처를 교차 확인한다.
- 사실(팩트)과 의견(해석)을 분리해 작성한다.
- 스토리텔링은 **조카와 대화하는 방식**을 기본 패턴으로 사용한다.
  - 짧은 대화(질문→설명→비유→정리)로 독자의 이해를 돕는다.
  - 과장/선동/공포 유도 표현은 금지한다.
- SEO 구조를 유지한다.
  - H1, 메타설명, H2/H3, FAQ, 태그, 내부링크 앵커 포함
  - 모바일 가독성을 위해 짧은 문단과 명확한 소제목 사용
